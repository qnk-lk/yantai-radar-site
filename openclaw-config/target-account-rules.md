# 目标客户识别规则（仅烟台 / 青岛）

## 当前客户拓展顺序

1. 烟台本地制造企业、园区、工厂、项目主体
2. 青岛制造企业、园区、工厂、项目主体

## 烟台优先目标客户

优先关注：

- 烟台本地制造业企业
- 烟台工业园区、开发区、产业园
- 烟台本地智能工厂、数字化车间、仓储升级、质量升级项目
- 烟台本地招聘 MES / WMS / QMS / 工厂IT / 信息化经理岗位的企业
- 烟台本地有扩产、建厂、技改、项目升级动作的企业

## 青岛扩展客户

重点关注：

- 青岛制造企业
- 青岛工业园区和智能制造示范项目
- 青岛厂商案例和项目招标
- 青岛高质量招聘和数字化项目信号

## 优先行业

- 装备制造
- 汽车与零部件
- 食品饮料
- 医药与医疗器械
- 化工与新材料
- 电子制造
- 机械加工
- 仓储物流相关制造企业

## 值得跟进的组织信号

- 招聘信息化经理
- 招聘 MES / WMS / QMS 顾问或实施岗位
- 招聘工厂IT、数字化项目经理、质量信息化岗位
- 发布系统采购、招标、中标信息
- 官网新闻出现“上线”“签约”“验收”“升级”“改造”等动作
- 园区、协会、政策名单中反复出现的重点企业
