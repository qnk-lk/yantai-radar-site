# 目标客户识别规则（烟台为核心）

## 当前客户拓展顺序
1. 烟台本地制造企业、园区、工厂、项目主体
2. 青岛、威海、潍坊等胶东半岛扩展区域
3. 济南、淄博、东营、济宁、临沂等山东重点制造城市
4. 山东其他地区的机会型高价值对象

## 烟台优先目标客户
优先关注：
- 烟台本地制造业企业
- 烟台工业园区、开发区、产业园
- 烟台本地智能工厂、数字化车间、仓储升级、质量升级项目
- 烟台本地招聘 MES / WMS / QMS / 工厂IT / 信息化经理岗位的企业
- 烟台本地有扩产、建厂、技改、项目升级动作的企业

## 胶东半岛扩展客户
重点关注：
- 青岛、威海、潍坊的制造企业
- 胶东地区工业园区和智能制造示范项目
- 胶东地区厂商案例和项目招标
- 胶东地区高质量招聘和数字化项目信号

## 山东补充区域客户
重点关注：
- 明确项目招标 / 采购 / 中标
- 明确客户案例落地
- 明确系统建设和岗位招聘信号
- 不追求铺量，优先强线索

## 优先行业
- 装备制造
- 汽车与零部件
- 食品饮料
- 医药与医疗器械
- 化工与新材料
- 电子制造
- 机械加工
- 仓储物流相关制造企业

## 值得跟进的组织信号
- 招聘信息化经理
- 招聘 MES / WMS / QMS 顾问或实施岗位
- 招聘工厂IT、数字化项目经理、质量信息化岗位
- 发布系统采购、招标、中标信息
- 官网新闻出现“上线”“签约”“验收”“升级”“改造”等动作
- 园区、协会、政策名单中反复出现的重点企业

## 当前保留策略
### 烟台
- 优先保留 A / B 级线索
- C 级线索在主体明确时也可暂存观察

### 胶东半岛
- 优先保留 A / B 级线索

### 山东其他重点城市
- 优先保留 A 级线索
- B 级线索仅在价值较高时保留

### 其他地区
- 默认只保留 A 级强线索

## 跟进建议逻辑
- 烟台本地：优先建立深度名单，便于线下或近距离跟进
- 胶东半岛：作为第二层扩展区域，重点保留高价值对象
- 山东其他：作为远期扩展池，不平均分配精力
