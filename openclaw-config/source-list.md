# 首批信息源清单（新加坡可运行版）

## 项目范围

- 地区：烟台、青岛
- 主题：MES、WMS、QMS
- 目标：观察行业现况、项目线索、客户案例、招标、招聘、政策、竞争变化

## 使用原则

- 优先使用新加坡出口可稳定访问的来源
- 不优先抓首页
- 优先抓搜索结果定位后的具体文章页、公告页、职位页、案例页
- 明确存在区域限制的站点，不进入自动化主流程
- 发生 403 区域限制时，不重复重试

## 来源 1：搜索引擎定向检索

- 类型：入口型来源
- 优先级：最高
- 主要用途：
  - 定位政府、园区、招标、案例、招聘的具体页面
- 使用方式：
  - 先通过关键词检索，定位可访问的具体页面
  - 再抓正文
- 推荐检索词：
  - site:\*.gov.cn 烟台 MES
  - site:\*.gov.cn 烟台 智能工厂
  - site:\*.gov.cn 烟台 数字化车间
  - site:\*.gov.cn 青岛 MES
  - site:\*.gov.cn 青岛 智能工厂
  - site:\*.gov.cn 青岛 数字化车间
  - site:\*.gov.cn 烟台 仓储管理系统 招标
  - site:\*.gov.cn 青岛 质量管理系统 招标
  - site:\*.com 烟台 MES 案例
  - site:\*.com 青岛 WMS 案例
  - site:\*.com 烟台 QMS 招聘
  - site:\*.com 青岛 MES 招聘

## 来源 2：山东省及地市政府 / 工信 / 园区文章页

- 类型：政策、通知、项目、示范案例
- 优先级：高
- 主要用途：
  - 找智能工厂、数字化车间、工业互联网、质量提升、仓储升级相关政策和项目
- 抓取重点：
  - 文章详情页
  - 通知公告详情页
  - 项目申报详情页
  - 示范名单详情页
- 不建议：
  - 直接抓首页

## 来源 3：官方招采 / 公告详情页

- 类型：招标、采购、中标
- 优先级：高
- 主要用途：
  - 找 MES、WMS、QMS 项目采购和招标线索
- 抓取重点：
  - 单条公告详情页
  - 中标结果详情页
  - 搜索结果跳转后的原始公告页
- 不建议：
  - 直接抓招采首页

## 来源 4：招聘平台职位页

- 类型：招聘
- 优先级：高
- 主要用途：
  - 找企业推进信息化建设的组织信号
- 优先岗位词：
  - MES
  - WMS
  - QMS
  - 信息化经理
  - 工厂IT
  - 制造执行系统
  - 仓储系统
  - 质量系统
  - 实施顾问
  - 数字化项目经理
- 优先城市：
  - 烟台
  - 青岛

## 来源 5：软件厂商案例详情页

- 类型：案例、签约、上线、验收
- 优先级：高
- 主要用途：
  - 找烟台或青岛客户案例
  - 找签约、上线、验收、实施新闻
- 抓取重点：
  - 客户案例详情页
  - 新闻详情页
- 不建议：
  - 抓厂商首页
  - 抓纯品牌页

## 来源 6：制造企业官网新闻详情页

- 类型：企业动态
- 优先级：中高
- 主要用途：
  - 找企业数字化转型、工厂改造、质量体系、仓储自动化等信号
- 注意：
  - 若未明确 MES / WMS / QMS，则只能作为中等线索或背景

## 来源 7：协会 / 展会 / 论坛详情页

- 类型：会议、论坛、协会活动
- 优先级：中
- 主要用途：
  - 找区域产业活跃度
  - 找高频出现的厂商和企业

## 来源 8：行业媒体文章详情页

- 类型：行业资讯
- 优先级：中
- 主要用途：
  - 发现线索
  - 辅助交叉验证
- 要求：
  - 优先带企业名、项目名、时间、地点
  - 转载稿默认降级
