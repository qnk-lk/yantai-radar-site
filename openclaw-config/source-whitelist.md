# 来源白名单（新加坡可运行版）

## 目标
保留“在新加坡服务器上更稳定可抓取”的来源，降低区域限制与首页失败带来的噪音。

## 总原则
- 首页不是优先抓取对象
- 具体文章页、公告页、案例页、职位页优先级高
- 搜索结果页只用于定位
- 区域受限来源不进入自动化主流程

## A 类来源：优先级最高

### 1. 原始文章详情页
适合：
- 政策
- 项目通知
- 园区动态
- 示范案例
要求：
- 原始发布时间清晰
- 主体清晰
- 地区清晰
- 页面正文可直接抓取

### 2. 招标 / 中标 / 采购公告详情页
适合：
- 项目招标
- 中标结果
- 采购需求
要求：
- 项目名称明确
- 主体明确
- 时间明确
- 最好能识别 MES / WMS / QMS 或相关系统范围

### 3. 招聘职位正文页
适合：
- MES / WMS / QMS 相关岗位
- 信息化经理、工厂IT、实施顾问等岗位
要求：
- 企业名称明确
- 地点明确
- 岗位明确
- 发布时间尽量明确

### 4. 软件厂商案例详情页
适合：
- 山东客户案例
- 签约、上线、验收新闻
要求：
- 客户名称明确
- 地区明确
- 发布时间明确

## B 类来源：可用但要谨慎

### 5. 政府 / 园区栏目页
作用：
- 发现可读文章链接
注意：
- 栏目页通常不是最终证据

### 6. 搜索结果页
作用：
- 定位具体页面
注意：
- 只作导航，不作最终证据

### 7. 制造企业新闻栏目页
作用：
- 快速发现企业近期动态
注意：
- 需进一步进入详情页

### 8. 行业媒体文章页
作用：
- 发现线索
- 补充验证
注意：
- 转载稿默认降级

## C 类来源：默认降级

### 9. 各类首页
包括：
- 政府首页
- 招采首页
- 厂商首页
- 企业首页
原因：
- 信息密度低
- 抓取稳定性差
- 证据价值弱

### 10. 区域受限来源
包括：
- zhiliaobiaoxun.com
处理方式：
- 当前不进入自动化主流程
- 不重复尝试抓取
- 如确有需要，后续人工核查

### 11. 强 JS / 反爬严重页面
处理方式：
- 不反复重试
- 用搜索定位其他页面
- 切换到其他白名单来源

## 当前优先顺序
1. 文章详情页 / 公告详情页 / 招标详情页
2. 职位正文页 / 案例详情页
3. 栏目页找链接
4. 搜索结果定位页面
5. 首页默认不抓
