# 潜在客户名单模板

## 目标
把日报中出现的企业、项目主体、园区、厂商线索沉淀成可持续跟踪的潜在客户名单，而不是只停留在当天摘要。

## 使用原则
- 只记录值得继续跟进的对象
- 每个对象尽量记录“主体 + 地区 + 信号 + 当前判断 + 下一步动作”
- 不追求一次写全，先让名单可用
- 弱线索也可以记录，但要明确线索等级和待核实项

## 推荐字段

### 1. 基本信息
- 名称：
- 类型：制造企业 / 软件厂商 / 集成商 / 园区 / 协会 / 政策主体 / 其他
- 所在地：山东省 + 城市
- 行业：装备制造 / 汽车 / 食品 / 医药 / 化工 / 电子 / 机械 / 其他
- 官网或主页：

### 2. 当前信号
- 当前主题：MES / WMS / QMS / 多模块 / 待判断
- 当前信号类型：招标 / 招聘 / 案例 / 政策 / 园区项目 / 客户转型 / 待判断
- 最近一次观察时间：
- 最近一次来源：
- 当前摘要：

### 3. 销售判断
- 线索等级：A / B / C / D
- 可信度：高 / 中 / 低
- 项目阶段：招标 / 规划 / 招聘信号 / 实施中 / 已上线 / 待核实
- 销售价值判断：高 / 中 / 低
- 是否值得持续跟踪：是 / 否
- 推荐动作：继续跟 / 观察 / 放弃

### 4. 待核实项
- 待核实 1：
- 待核实 2：
- 待核实 3：

### 5. 下一步动作
- 下一步动作 1：
- 下一步动作 2：
- 下一步动作 3：

## 单个客户记录示例

### 客户名称：示例制造企业
- 类型：制造企业
- 所在地：山东省青岛市
- 行业：装备制造
- 官网或主页：待补充

#### 当前信号
- 当前主题：MES
- 当前信号类型：招聘
- 最近一次观察时间：2026-04-09
- 最近一次来源：智联招聘
- 当前摘要：出现 MES 实施顾问 / 工厂信息化相关岗位，疑似新项目或项目扩张信号

#### 销售判断
- 线索等级：B
- 可信度：中
- 项目阶段：招聘信号
- 销售价值判断：中
- 是否值得持续跟踪：是
- 推荐动作：继续跟

#### 待核实项
- 是否有明确工厂或项目主体
- 是否还有其他系统相关岗位
- 是否能在官网或新闻中找到数字化建设信息

#### 下一步动作
- 核验招聘页发布时间和地点
- 查询企业官网新闻或招标信息
- 查是否存在 MES / 智能工厂 / 车间管理相关公开信息

## 名单维护规则
- A / B 级线索优先进入名单
- C 级线索仅在主体明确时保留
- D 级线索默认不入库
- 每次日报后，如果出现新企业或新项目主体，应判断是否补入名单
- 同一对象后续有新动态时，追加更新时间和最新信号
