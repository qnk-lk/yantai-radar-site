# 核验清单

## 项目招标 / 采购线索核验
- 是否有原始招标页面
- 是否有明确项目名称
- 是否有明确主体单位
- 是否能确认项目与 MES / WMS / QMS 相关
- 是否有发布时间和地区

## 招聘线索核验
- 是否有企业名称
- 是否有岗位名称
- 是否有工作地点
- 是否有发布时间
- 是否能判断岗位与 MES / WMS / QMS 或工厂信息化直接相关

## 厂商案例核验
- 是否有客户名称
- 是否有山东地区信息
- 是否有发布时间
- 是否有项目动作描述（上线、签约、实施、验收）

## 政策 / 园区线索核验
- 是否来自原始政府或园区页面
- 是否有明确发布时间
- 是否与制造业数字化项目直接相关
- 是否能转化为潜在线索或市场判断
