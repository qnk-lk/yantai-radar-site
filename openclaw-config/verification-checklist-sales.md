# 销售线索核验清单

## 招聘线索核验
- 企业名称是否明确
- 工作地点是否明确到山东城市
- 岗位是否直接指向 MES / WMS / QMS / 工厂信息化
- 是否有发布时间
- 是否能判断是新项目、扩张还是补位
- 是否值得进入潜在客户名单

## 招标 / 采购线索核验
- 是否有原始招标页面
- 项目名称是否明确
- 采购主体是否明确
- 是否能判断是 MES / WMS / QMS 还是相关数字化项目
- 是否有发布时间
- 是否处于可销售接触阶段

## 客户案例核验
- 客户名称是否明确
- 是否在山东
- 是否有项目动作（签约、上线、验收、实施）
- 是否能判断需求方向
- 是否说明项目阶段
- 是否对竞对分析有价值

## 政策 / 园区线索核验
- 是否来自政府、园区、工信、协会原始页面
- 是否有明确发布时间
- 是否可转化为具体企业或项目名单
- 是否能形成下一步客户挖掘方向

## 输出前最后判断
- 这条信息是否值得销售跟进
- 这条信息更像项目机会还是市场背景
- 下一步具体该查什么
